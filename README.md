This game is designed to be played on your terminal. To play it, **YOU MUST** first install node.js and npm.

Node.js is a program that expands javascripts uses to more than just web development. It's mainly used for server-side development.
NPM stands for Node Package Manager. Should be self-explanatory what it is for.

# Installation & Execution Instructions

Installing and running the program is fairly simple, but in case you need a run down here are the instructions.
<br>
<br>
### 1. Clone the repository (you probably already did this but just in case you didn't here's the command) or download the zip file: https://github.com/PianoZombie/Russian-Roulette/archive/refs/heads/main.zip
```shell
git clone https://github.com/PianoZombie/Russian-Roulette.git [target_directory]
```


### 2. Open the terminal (if you haven't already) and cd into the repository
```shell
cd C:\The\Path\To\The\Repository
```

### 3. Install the program's dependencies
```shell
npm install
```

### 4. Run the program (if you're on Windows, make sure you run your terminal w/ administrator!)
```shell
node russian-roulette.js
```

Congratulations! You have sucessfully executed the program!
